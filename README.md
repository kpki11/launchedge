## What is LaunchEdge?

LaunchEdge is a suite of business tools built specifically for Indian MSMEs, startups, and entrepreneurs — people who run real businesses in the real India.

Most business software is built for Silicon Valley. LaunchEdge is built for **Bharat** — offline-first, no-code, mobile-first, and completely free to start.

## 📲 LaunchEdge Labs — Free ERP App

> **Your data never leaves your phone. We store nothing. Ever.**

The flagship product. A fully offline Android app that lets any small business owner build their own ERP — no coding, no subscriptions, no cloud.

**Features:**
- 📊 Import Excel / CSV files → instant smart tables
- 🔗 Link tables together (orders ↔ inventory ↔ payments)
- 📈 Live charts that update as your data changes
- 🗓️ Calendar view for any date field
- 📄 Auto-generate PDF reports
- ⚡ Workflow automation — set conditions, trigger actions
- 📁 Google Drive import support
- 🏗️ 50+ ready-to-use templates (billing, HR, inventory, expenses)
- 🔒 Zero data collection. Zero internet required.

## 🌐 Website

The LaunchEdge website is built with pure HTML, CSS, and JS — no frameworks, no build tools. Fast, offline-capable, mobile-first.

**Pages:**
- `index.html` — Homepage
- `pages/labs.html` — Labs product page
- `pages/copilot.html` — CoPilot AI page
- `pages/talent.html` — Talent network page
- `pages/about.html` — About LaunchEdge
- `pages/contact.html` — Contact form
- `pages/privacy.html` — Privacy policy
- `pages/terms.html` — Terms of service

**Stack:** HTML5 · CSS3 · Vanilla JS · Google Fonts · Formspree (contact form) · GoatCounter (privacy-friendly analytics)
